# Shai-Hulud-2-Checker
A simple script that checks repositories for affected Packages.

It has no external dependancies and is only 42 Lines long.
It prints out the affected packages as an Array.
It shows packages that have the same name and if the packages are matching the version number.
It checks node modules in package-lock.json as well as their dependancies.

Get compromised-packages.json from https://github.com/gensecaihq/Shai-Hulud-2.0-Detector
